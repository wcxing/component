页面结构说明

<!DOCTYPE html>
<html lang="zh-cn">
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Hello world!</title>
    <!-- 框架样式 -->
    <link href="css/interface.css" rel="stylesheet">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <!-- 在此处挥洒你的创意 -->
    <!-- jQuery (框架中的的Javascript组件依赖于jQuery) -->
    <script src="//code.jquery.com/jquery-1.11.0.min.js"></script>
    <!-- 框架 Javascript组件 -->
    <script src="js/interface.min.js"></script>
  </body>
</html>

页面说明

<!DOCTYPE html>是用来声明文档的，页面必须要有此声明，并且位于页面顶部，无此声明或者声明多次会导致在部分浏览器中存在难以解决和定位的兼容bug.

<meta http-equiv="X-UA-Compatible" content="IE=edge">禁用IE 的兼容模式，部分浏览器中默认会以ie兼容模式渲染页面，从而产生意外的bug。因此需要禁用

<meta name="viewport" content="width=device-width, initial-scale=1">响应式代码判断的依据，如果无此设置，在手机浏览器上会将整个页面缩小，放大后需要用手指上下左右方向滑屏才能看到局部。设置后，响应式代码才会正常工作
