文件目录结构

下载预编译压缩包解压后将会得到以下目录结构：

Interface/
├── pub-ui/
│    │
│    ├── css/
│    │   ├── interface.css                //编译后css文件
│    │   ├── interface.min.css            //压缩版本css文件，建议引用压缩版本
│    │   ├── font-awesome.css             //字体图标css文件
│    │   ├── font-awesome.min.css         //压缩版本字体图标css文件
│    │   └── style.css                    //自定义样式文件
│    ├── js/
│    │   ├── interface.js                 //编译后js文件
│    │   ├── interface.min.js             //压缩版本js文件，建议引用压缩版本
│    │   ├── jquery.min.js                //jquery文件
│    │   └── ieonly/                      //针对IE8及早期版本兼容处理的js文件
│    ├── fonts/                           //框架所需字体文件
│    └── plugins/                         //第三方库文件
└── pub-page/
     └── demo.html                        //基础页面结构demo
