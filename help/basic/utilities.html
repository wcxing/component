框架提供了众多实用的辅助类。任何时候只需要加入对应的CSS类名即可。虽然辅助类使用起来非常方便，但也要避免滥用。使用之前应该了解以下要点。

  • 尽量少用或不用辅助类，如果有对应的控件或组件使用。
  • 当相同的场景需要大量使用辅助类时，这时应该考虑是否单独定义一个新的控件或组件。

字号文本

使用标签<small>或者CSS类名.small来使得文本字号比正常情况小一号。

小字号文本，small text. 
<small>小字号文本，small text.</small>
正常 14px <span class="font-sm">字号</span>
.font-sm 12px <span class="font-sm">字号</span>
.font-md 14px <span class="font-md">字号</span>
.font-big 18px <span class="font-big">字号</span>
.font-lg 24px <span class="font-lg">字号</span>
.font-huge 37px <span class="font-huge">字号</span>

盒模型尺寸

一个元素完整的尺寸饱和上图上的width、height、padding、border-with、margin。框架预定义了margin和padding的尺寸大小，分别加0、sm、md、big、lg、huge，可以单独的定义某一方向的值，也可以4个方向的同时定义。比如.pd-0为元素的上下左右的内边距都为0，常用来覆盖默认的内边距，也可以单独设置为.pdl-0等

0 0 .pd-0 .pdl-0 .pdt-0 .pdr-0 .pdb-0 .mg-0 .mgl-0 .mgt-0 .mgr-0 .mgb-0
sm 5px .pd-sm .pdl-sm .pdt-sm .pdr-sm .pdb-sm .mg-sm .mgl-sm .mgt-sm .mgr-sm .mgb-sm
md 10px .pd-md .pdl-md .pdt-md .pdr-md .pdb-md .mg-md .mgl-md .mgt-md .mgr-md .mgb-md
big 15px .pd-big .pdl-big .pdt-big .pdr-big .pdb-big .mg-big .mgl-big .mgt-big .mgr-big .mgb-big
lg 25px .pd-lg .pdl-lg .pdt-lg .pdr-lg .pdb-lg .mg-lg .mgl-lg .mgt-lg .mgr-lg .mgb-lg
huge 50px .pd-huge .pdl-huge .pdt-huge .pdr-huge .pdb-huge .mg-huge .mgl-huge .mgt-huge .mgr-huge .mgb-huge

禁用文本换行

            

            

                

                    使所有文本在一行显示。超出的部分将会被隐藏。只需要使用.text-nowrap或者.nobr。

                
<h2 class="text-nowrap">...</h2>

            

        

        

            

                
文本超出省略

            

            

                
使所有文本在一行显示。超出的部分会被截断，并且在文本末尾添加省略号。使用.text-ellipsis。

                
<h2 class="text-ellipsis">...</h2>

                

                    
并不是所有浏览器都支持此选项。

隐藏文本

            

            

                
隐藏文本能够使得标签内的文本不显示，通常用于图片代替文本。

                
<h1 class="text-hide">此处文本不会显示</h1>

            

        

        

            

                
显示或隐藏元素

            

            

                
提供多个辅助类来帮助切换内容显示。

                
.hide{
  display: none;
}
.hidden{
  display: none!important;
}
.show{
  display: block;
}
.showing{
  display: block!important;
}
.invisible{
  visibility: hidden;
}

                

                    .hidden和.showing具备更高的优先级，防止其他规则重写。.invisible仅仅隐藏元素，但元素内容所占的空间并不会清除。
