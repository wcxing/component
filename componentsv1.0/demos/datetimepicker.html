日期选择使用国产插件，可以自定义时间选择范围，使用比较灵活。

详细的参数使用，可以参考My97 Datepicker

默认

限制起止时间

前面的日期不能大于后面的日期且两个日期都不能大于 2020-10-01

--
<input id="mindate" type="text" class="form-control date-picker" onclick="WdatePicker({maxDate:'#F{$dp.$D(\'maxdate\')||\'2020-10-01\'}'})">
--
<input id="maxdate" type="text" class="form-control date-picker" onclick="WdatePicker({minDate:'#F{$dp.$D(\'mindate\')}',maxDate:'2020-10-01'})">
                        

注意:
两个日期的日期格式必须相同
$dp.$ 相当于 document.getElementById 函数。那么为什么里面的 ' 使用 \' 呢? 那是因为 " 和 ' 都被外围的函数使用了,故使用转义符 \ ,否则会提示JS语法错误。所以您在其他地方使用时注意把 \' 改成 " 或者 ' 来使用。
#F{$dp.$D(\'maxdate\')||\'2020-10-01\'} 表示当 maxdate 为空时, 采用 2020-10-01 的值作为最大值

前面的日期+3天 不能大于 后面的日期

--
<input type="text" class="Wdate" id="d4321" onFocus="WdatePicker({maxDate:'#F{$dp.$D(\'d4322\',{d:-3});}'})"/>
<input type="text" class="Wdate" id="d4322" onFocus="WdatePicker({minDate:'#F{$dp.$D(\'d4321\',{d:3});}'})"/>


使用 $dp.$D 函数 可以将日期框中的值,加上定义的日期差量:
两个参数: id={字符类型}需要处理的文本框的id值 , obj={对象类型}日期差量
日期差量用法:
属性y,M,d,H,m,s分别代表年月日时分秒
如 
为空时,表示直接取值,不做差量(示例4-3-1中的参数就是空的)
{M:5,d:7} 表示 五个月零7天
{y:1,d:-3} 表示 1年少3天
{d:1,H:1} 表示一天多1小时
