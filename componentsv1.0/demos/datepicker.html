Datepicker 是一款基于Query UI的日期选择器。您可以高度配置插件以及自定义日期格式和语言，限制可选择的日期范围，添加按钮和其他导航选项

详细的参数使用，可以参考 jQuery UI Datepicker API

更多案例，可以参考 jQuery UI Datepicker Examples

用法

默认情况下，当相关的文本域获得焦点时，在一个小的覆盖层打开日期选择器。对于一个内联的日历，只需简单地将日期选择器附加到 div 或者 span 上。

默认

$("#J_datepicker").datepicker({
    showOtherMonths: true,      //默认：false  是否显示其他月份的日期
    showButtonPanel: true,      //默认：false  是否显示一个 选择当天日期"Today"按钮，显示一个关闭日历"Close"按钮
    changeMonth: true,          //默认：false  是否显示可选择月份
    changeYear: true,           //默认：false  是否显示可选择年份
    dateFormat: "yy-mm-dd",     //默认：yy-mm-dd 默认和选择语言有关  格式化日期
    minDate: -1,                //限制日期范围,通过 minDate 和 maxDate 选项限制可选择的日期范围。设置起止日期为实际的日期（new Date(2009, 1 - 1, 26)），或者为与今天的一个数值偏移（-20），
    maxDate: "+1M +10D"         //或者为一个周期和单位的字符串（'+1M +10D'）。如果设置为字符串，使用 'D' 表示天，使用 'W' 表示周，使用 'M' 表示月，使用 'Y' 表示年。
});
