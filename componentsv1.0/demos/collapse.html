折叠（Collapse）插件可以很容易地让页面区域折叠起来。无论您用它来创建折叠导航还是内容面板，它都允许很多内容选项。

简单折叠

用a标签href触发
简单折叠 
<a class="btn btn-primary" role="button" data-toggle="collapse" href=".collapseExample">
    用a标签href触发
</a>
<button class="btn btn-primary" type="button" data-toggle="collapse" data-target=".collapseExample">
    用button标签data-target触发
</button>
<div class="collapseExample collapse in" id="">
  <div class="well">
    简单折叠
  </div>
</div>

用法

该组件的js已合并到interface.js，如单独使用可引用collapse.js

.collapse隐藏内容，.collapse .in显示内容

调用方式一：仅仅通过向页面元素添加 data-toggle="collapse" 和 data-target 就可以为其赋予控制可折叠页面元素的能力。 data-target 属性接受一个CSS选择器作为其控制对象。请确保为可折叠页面元素添加 .collapse 。 如果你希望可折叠页面元素的默认状态是展开的，请添加 .in 。 为了给一组可折叠页面元素添加控制器，添加 data-parent=".selector",并确保这一组可折叠被名为.selector所包裹即可，如上面实例中的data-parent=".panel-group"。

调用方式二：手动触发 $(".collapse").collapse()

方法和事件

方法 使用 描述
.collapse(options) $('#myCollapse').collapse({ toggle: false }) 赋予页面元素可折叠功能。接受一个可选的object作为参数。
.collapse('toggle') $('#myCollapse').collapse('toggle') 展示或隐藏一个可折叠的页面元素。
.collapse('show') $('#myCollapse').collapse('show') 展示一个可折叠的页面元素。
.collapse('hide') $('#myCollapse').collapse('hide') 隐藏一个可折叠的页面元素。
事件 使用 描述
show.bs.collapse $('#myCollapse').on('show.bs.collapse', function () { // do something… }) 当 show 方法被调用时，此事件将被立即触发
shown.bs.collapse $('#myCollapse').on('shown.bs.collapse', function () { // do something… }) 当可折叠页面元素显示出来之后（同时CSS过渡效果也已执行完毕），此事件被触发。
hide.bs.collapse $('#myCollapse').on('hide.bs.collapse', function () { // do something… }) 当 hide 方法被调用时，此事件将被立即触发。
hidden.bs.collapse $('#myCollapse').on('hidden.bs.collapse', function () { // do something… }) 当可折叠页面元素隐藏之后（同时CSS过渡效果也已执行完毕），此事件被触发。
