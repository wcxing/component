单选和复选的input外层嵌套<label>标签,可以使点击文字的时候出发选中操作，此细节会提升用户体验！

<div class="checkbox">
  <label>
    <input type="checkbox"> 复选框
  </label>
</div>
<div class="checkbox disabled">
  <label>
    <input disabled type="checkbox"> 被禁用的复选框
  </label>
</div>
 
<div class="radio">
  <label>
    <input type="radio" name="radioOptionsExample" id="radioOptionsExample1"> 单选框1
  </label>
</div>
<div class="radio">
  <label>
    <input type="radio" name="radioOptionsExample" id="radioOptionsExample2"> 单选框2
  </label>
</div>
<div class="radio disabled"> 
  <label>
    <input type="radio" value="" disabled="disabled" name="radioOptionsExample" id="radioOptionsExample3"> 被禁用的单选框 
  </label>
</div>
 
<textarea class="form-control" rows="3" placeholder="可以输入多行文本"></textarea>
 
<input type="file" class="form-control" value="">

用定位方式实现的文件选择，设置file输入框的透明度为0，实际点击的也是这个file输入框，必须使用form-control-file此结构才会有效果

<div class="form-control-file">
    <label>
        <input type="file" value="" class="form-control">
        <input type="text" class="form-control" placeholder="选择文件">
        <button class="btn btn-warning">浏览</button>
    </label>
</div>

  <option value="orange">桔子</option>
  ...
</select>
